# Customer Analytics: CLV vs Acquisition Cost Analysis

This repository contains a Seaborn-based visualization analyzing the relationship between Customer Lifetime Value (CLV) and Customer Acquisition Cost (CAC) for strategic decision-making.

## Files
- `chart.py` — Generates the Seaborn scatterplot
- `chart.png` — Output visualization (512×512 pixels)
- `README.md` — Project documentation
